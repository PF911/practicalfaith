App Features
What this app includes
• Multiple quiz categories and three difficulty levels
• Resume Last Quiz so you can pick up where you left off
• Two-try answer system before the correct answer is revealed
• Scripture reference shown after each question
• “Click to see the verse” popup with the full verse text
• High Scores and My Stats tracking
• In-quiz streak meter for correct answers in a row
• Category progress bars so you can see long-term progress
• Sound effects with Sound On / Off control
• Perfect-score celebration with confetti
